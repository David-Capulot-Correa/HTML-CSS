# 🧮 Calculadora 2DEGJ

## 📌 Sobre o projeto

A **Calculadora 2DEGJ** é uma aplicação web simples e funcional desenvolvida para realizar operações matemáticas básicas de forma rápida e intuitiva.

O projeto foi criado com o objetivo de praticar conceitos de desenvolvimento web, utilizando HTML, CSS e JavaScript.

---

## 🚀 Funcionalidades

* ➕ Soma
* ➖ Subtração
* ✖️ Multiplicação
* ➗ Divisão (com tratamento para divisão por zero)
* 🔢 Resto da divisão
* 🧹 Limpar dados rapidamente
* ⚠️ Validação de entrada de números

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 📷 Preview

O sistema possui:

* Interface simples e organizada
* Formulário para inserção de números
* Seleção de operação matemática
* Resultado exibido dinamicamente na tela

---

## 🎯 Objetivo

Este projeto foi desenvolvido com fins acadêmicos para:

* Praticar lógica de programação
* Trabalhar com manipulação do DOM em JavaScript
* Criar uma aplicação funcional e interativa
